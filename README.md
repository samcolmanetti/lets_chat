# Let's Chat
Let's chat is a simple chat application to discuss various topics with friends.

![Lets_ChatP_Preview](https://github.com/samsoares/lets_chat/blob/master/resources/Preview.PNG)

## Installation
### Server
```
npm i
```
### Client 
```
cd client
npm i
```

## Deployment instructions
### Server
```
npm start
```
### Client 
```
cd client
npm start
```
